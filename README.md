# 🍽️ RestoranTakip
### Online Restaurant Order Tracking System

> Built with **C#** · **Windows Forms** · **Visual Studio**

---

## 📖 Overview

**RestoranTakip** is a Windows desktop application built with C# and WinForms that enables restaurant staff to manage orders, track tables, and monitor the entire service workflow from a single, easy-to-use interface.

Designed for small to medium-sized restaurants, the system eliminates the need for paper-based order tracking and reduces errors by centralizing all order and table management operations.

---

## ✨ Features

| Feature | Description |
|---|---|
| **Order Management** | Create, update, and track restaurant orders in real time |
| **Table Tracking** | Monitor table status and customer seating |
| **Menu Management** | Add, edit, and categorize menu items with prices |
| **Order Status Updates** | Live status tracking from placement to delivery |
| **Bill Generation** | Automatically calculate and generate customer bills |
| **User Interface** | Clean WinForms desktop UI for restaurant staff |

## 🚀 Getting Started

### Prerequisites

- Visual Studio 2019 or later *(Community edition is free)*
- .NET Framework or .NET SDK *(check the `.sln` for the target version)*
- Windows OS *(WinForms is Windows-only)*

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/muhammedsali/RestoranTakip.git
   ```

2. **Open the solution in Visual Studio**
   ```bash
   cd RestoranTakip
   start RestoranTakip.sln
   ```

3. **Build and run**

   Press `F5` in Visual Studio or use **Build → Start Debugging**.

## 📋 Usage

- Launch the application from Visual Studio or the compiled executable.
- Use the main dashboard to view active tables and open orders.
- Create new orders by selecting a table and adding menu items.
- Update order status as items are prepared and served.
- Generate a bill when the customer is ready to pay.
